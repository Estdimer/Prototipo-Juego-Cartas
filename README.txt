Unity Version: 6000.0.32f1

Prototipo de juego de cartas
Objetivo: -Jugar personajes en el tablero
           para derrotar a los enemigos de la piramide
          -Editar cartas en el editor de cartas para
           ajustar el balance del juego

Cómo abrir el codigo:
1. Abrir Unity Hub
2. Seleccionar "Add" => Add Proyect From Disk, y elegir carpeta JuegoCartas
3. En Assets/Scripts se encuentra los codigos utilizados

Como abrir aplicacion Computador:
1. Se necesita tener windows instalado
2. Descargar carpeta EjecutableComputador 
3. iniciar JuegoCartas.exe

Como abrir aplicacion Celular:
1. Se necesita un sistema android (no compatible con todas las resoluciones) 
2. Abrir carpeta APK Juego De Cartas
3. Instalar JUEGO DE CARTAS.apk
4. Abrir aplicacion Juego De Cartas


Requerimientos funcionales

RF-01
Nombre: La aplicación debe permitir a los usuarios visualizar las cartas en su mano. 
Descripción: La aplicación permitirá a los usuarios ver las cartas que poseen actualmente, incluyendo el diseño visual de cada una, sus estadísticas y habilidades.

RF-02
Nombre: La aplicación debe permitir a los usuarios mover cartas a la pantalla.
Descripción: La aplicación permitirá a los usuarios seleccionar una carta de su mano y moverla a través de la pantalla y ver si hay lugares en los que se pueda interactuar.

RF-03
Nombre: La aplicación debe permitir a los usuarios colocar cartas en el campo de juego.
Descripción: La aplicación permitirá a los usuarios seleccionar una carta de su mano y posicionarla en un área del campo disponible, validando si es jugable según el turno y el costo.

RF-04
Nombre: La aplicación debe permitir a los usuarios comprar cartas desde la tienda del juego.
Descripción: La aplicación permitirá a los usuarios acceder a una tienda virtual donde podrán intercambiar monedas del juego por nuevas cartas, mostrando detalles y rareza antes de confirmar la compra.

RF-05
Nombre: La aplicación debe permitir a los usuarios seleccionar un jefe dentro del tablero.
Descripción: La aplicación permitirá a los usuarios seleccionar un jefe dentro del tablero de entre las opciones de jefes disponibles.

RF-06
Nombre: La aplicación debe permitir a los usuarios iniciar batallas contra jefes.
Descripción: La aplicación permitirá a los usuarios presionar un botón para entrar en combate con un jefe seleccionado previamente.

RF-07
Nombre: La aplicación debe permitir a los usuarios visualizar las estadísticas de las cartas.
Descripción: La aplicación mostrará un tooltip al pasar el cursor o tocar una carta, desplegando información como ataque, defensa y habilidades.

RF-08
Nombre: La aplicación permitirá a los usuarios ver un resaltado visual sobre cartas jugables y las zonas del campo donde puedan ser colocadas.
Descripción: La aplicación mostrará un efecto de brillo o resaltado sobre las cartas jugables y las zonas válidas del campo al interactuar con ellas, facilitando la identificación de acciones posibles.

Requerimientos no funcionales

RNF-01
Nombre: La aplicación debe cargar las cartas en menos de 2 segundos al iniciar una partida.
Descripción: La aplicación garantizará una carga rápida y eficiente del mazo inicial para asegurar una experiencia fluida.

RNF-02
Nombre: La aplicación debe asegurar que las animaciones del campo de batalla se ejecuten sin interrupciones visibles.
Descripción: La aplicación garantizará un rendimiento fluido de las animaciones de ataque, invocación y efectos visuales, incluso en dispositivos de gama media.

RNF-03
Nombre: La aplicación debe ser compatible con dispositivos móviles y de escritorio.
Descripción: La aplicación se desarrollará con soporte multiplataforma, adaptando la interfaz gráfica y controles a cada dispositivo.

RNF-04
Nombre: La aplicación debe asegurar un rendimiento mínimo de 30 fps durante el uso general.
Descripción: La aplicación garantizará  una tasa de refresco mínima de 30 cuadros por segundo durante las partidas, para asegurar una experiencia fluida en distintos dispositivos.
